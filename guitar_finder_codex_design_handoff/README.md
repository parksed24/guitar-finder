# Guitar Finder Codex Design Handoff

Copy these files into the root of your existing `guitar-finder` repository:

- `reference-prototype.html`
- `AGENTS.md`
- `design-parity-checklist.md`
- `CODEX_PROMPT.txt`

Then ask Codex to execute the contents of `CODEX_PROMPT.txt`.

The HTML file is the exact latest working prototype and should be treated as the visual source of truth.
